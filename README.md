## Getting Started

Clone the repository and install dependencies:

```bash
git clone <your-repo-url>
cd <your-repo-name>
npm install
```

Then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
